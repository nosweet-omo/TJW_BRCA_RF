# BRCA eRNA Analysis Project 

本项目包含用于分析 TCGA-BRCA 数据的 Python 代码，主要涉及随机森林 (Random Forest) 和卷积神经网络 (CNN) 模型在 eRNA 数据上的应用。

## 📂 数据集下载 

由于 GitHub 文件大小限制，本项目所需的完整数据集（包括 TCGA 临床数据、eRNA 表达矩阵等大文件）存储在外部云盘中。

**请务必下载数据并放置在项目目录中，否则代码无法运行。**
